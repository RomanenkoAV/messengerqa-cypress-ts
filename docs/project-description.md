# Описание проекта

Messenger QA — учебный fullstack-проект для демонстрации навыков автоматизации тестирования web-приложений.

Проект имитирует базовые функции мессенджера

- авторизация
- список чатов
- поиск чатов
- просмотр истории сообщений
- отправка сообщений
- редактирование и удаление собственных сообщений
- статусы сообщений
- входящие сообщения через Socket.IO
- REST API
- адаптивный интерфейс

## Что демонстрирует проект

- проектирование UI и API автотестов
- использование TypeScript
- работа с Cypress
- Page Object
- custom commands
- fixtures
- независимость тестов через test API
- проверка realtime-сценариев
- тестирование адаптивности
- CI для GitHub Actions и GitLab CI
- тестовая документация
