# Как рассказать о проекте на собеседовании

Проект представляет собой web-мессенджер с REST API и realtime-обновлениями через Socket.IO. Для него был подготовлен Cypress-фреймворк на TypeScript.

UI-набор покрывает авторизацию, поиск чатов, отправку, редактирование, удаление и получение входящих сообщений без перезагрузки. API-набор проверяет авторизацию, создание сообщений, валидацию и доступ без токена.

Для стабильности тестов используются data-testid, Page Object, custom commands, fixtures и отдельный test API для сброса состояния. Запуск автоматизирован в GitHub Actions и GitLab CI.

Главный риск продукта связан с рассинхронизацией REST-ответов, Socket.IO событий и состояния интерфейса. Поэтому отдельно проверяется отсутствие дублей и корректное обновление истории сообщений.
