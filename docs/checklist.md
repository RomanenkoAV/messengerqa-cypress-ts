# Чек-лист мессенджера

## Авторизация

- успешный вход
- неверный email
- неверный пароль
- пустые поля
- выход из аккаунта
- повторное открытие приложения с сохранённой сессией

## Чаты

- загрузка списка
- сортировка по последнему сообщению
- поиск по полному и частичному названию
- пустой результат поиска
- переключение между чатами
- сброс unread после открытия

## Сообщения

- отправка короткого текста
- отправка текста длиной 500 символов
- запрет пустого сообщения
- защита от HTML и script injection
- редактирование собственного сообщения
- удаление собственного сообщения
- запрет изменения чужого сообщения
- корректное время и статус
- realtime-доставка
- отсутствие дублей

## API

- 200 и 201 для успешных запросов
- 204 для удаления и mark read
- 400 для невалидного текста
- 401 без токена
- 403 при попытке изменить чужое сообщение
- 404 для неизвестного чата или сообщения
