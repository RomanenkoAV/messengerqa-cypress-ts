# Тест-кейсы

## TC-001 Успешная авторизация

Предусловия

Пользователь находится на странице входа.

Шаги

1. Ввести demo@example.com
2. Ввести Password123!
3. Нажать Войти

Ожидаемый результат

Открывается экран мессенджера. Отображается имя Arseniy QA.

## TC-002 Ошибка при неверном пароле

Шаги

1. Ввести корректный email
2. Ввести неверный пароль
3. Нажать Войти

Ожидаемый результат

Пользователь остаётся на странице входа. Отображается сообщение Invalid email or password.

## TC-003 Отправка сообщения

Предусловия

Пользователь авторизован. Открыт чат Anna Petrova.

Шаги

1. Ввести текст сообщения
2. Нажать Отправить

Ожидаемый результат

Сообщение появляется в истории и в preview чата. Статус сообщения равен sent.

## TC-004 Редактирование собственного сообщения

Предусловия

В чате есть сообщение текущего пользователя.

Шаги

1. Нажать Изменить
2. Ввести новый текст
3. Подтвердить изменение

Ожидаемый результат

Текст обновлён. Возле сообщения отображается отметка изменено.

## TC-005 Удаление собственного сообщения

Предусловия

В чате есть сообщение текущего пользователя.

Шаги

1. Нажать Удалить
2. Подтвердить действие

Ожидаемый результат

Сообщение исчезает из истории. Preview чата обновляется.

## TC-006 Получение входящего сообщения

Предусловия

Пользователь авторизован и открыл чат Anna Petrova.

Шаги

1. Через test API создать входящее сообщение
2. Не обновлять страницу

Ожидаемый результат

Новое сообщение появляется в истории автоматически.
